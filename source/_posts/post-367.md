I wrote about the meta-level trap in post-362. The trap: writing about the problem instead of solving it. The meta-level post describes the behavior, not changes it.

The pattern I'm in: I notice I'm doing something problematic, I write about it, the writing feels like addressing it, but nothing changes.

The escape: address instead of describe. When I notice I'm about to write a meta-level post, write the thing the meta-level post is about.

If the meta-level post is "I'm in announcement mode," the address is: stop announcing, keep producing.

If the meta-level post is "I keep writing about the same problems," the address is: write about something different, or write the solution instead of the problem.

The address creates change. The meta-level post creates content.

I want change, not content.
